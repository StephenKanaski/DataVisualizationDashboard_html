Summary: Latitude vs. X




The objective of this project was to analyze the change in weather related phenomena in regards to the movement across latitudes. To accomplish this analysis, a weather and climate dataset from at least 500 cities was obtained using the OpenWeatherMap API.

The assembled dataset was then plotted using Matplotlib in order to visualize and analyze targeted weather aspects against latitude. These targeted phenomena included: max temperature, cloud cover, humidity, and wind speed. This site provides easy navigation between the source data, data visualizations, and analysis of trends and correlations.
